# MKU Graduands Clearance System

A web-based Graduands Clearance System built for **Mount Kigali University (MKU)** to streamline the clearance process for graduands. The system allows students to submit their clearance requests, departments to approve or reject requests, and provides real-time status updates.

## Features
- Student registration and login
- Department-based admin login
- Submit clearance requests and upload necessary files
- Approve or reject clearance requests by departments
- Real-time status updates for students and admins
- View which departments have cleared or rejected
- Notifications for departments when a student responds to a rejection

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other Tools:** npm, Postman (for API testing)
